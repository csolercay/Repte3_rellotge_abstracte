# Rellotge Abstracte

Extensió per a Firefox creada amb **p5.js**, que representa el pas del temps de manera **abstracta** a través de punts i línies connectades.

## Característiques
- Representació visual del temps
- Controls d’interacció (slider i botó pausa)
- Dades persistents amb `storeItem()` i `getItem()`

## Instal·lació
1. Descarrega o clona aquest repositori.
2. A Firefox, obre `about:debugging#/runtime/this-firefox`
3. Clica “Load Temporary Add-on” i selecciona `manifest.json`
